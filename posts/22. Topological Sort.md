title: "Topological Sort"
date: "2024-07-09"
description: "Topological Sort of a Directed Acyclic Graph (DAG) is a linear ordering of its vertices such that for every directed edge u -> v, vertex u comes before v in the ordering."
language: "python"
categories: [ "python", "graph" ]
---

```python
def topologicalSort(graph, vertices: list):
    permanent = set()
    temporary = set()
    stack = []

    def visit(u):
        if u in permanent:
            return True
        if u in temporary:
            return False
        temporary.add(u)
        for v in graph.get(u, []):
            if not visit(v):
                return False
        temporary.remove(u)
        permanent.add(u)
        stack.append(u)
        return True

    dag = True
    while dag and vertices:
        vertice = vertices.pop()
        if vertice not in permanent:
            dag = visit(vertice)

    return stack[::-1] if dag else "Not a DAG. Topological Sort not possible."

# Test case 1
graph1 = {5: [11], 11: [2, 9, 10], 2: [], 7: [8], 8: [9], 9: [], 3: [10], 10: [8]}
vertices1 = [5, 11, 2, 7, 8, 9, 3, 10]
print(topologicalSort(graph1, vertices1))  # Expected: A topologically sorted order

# Test case 2 (with a cycle)
graph2 = {1: [2], 2: [1]}
vertices2 = [1, 2]
print(topologicalSort(graph2, vertices2))  # Expected: "Not a DAG. Topological Sort not possible."
```
